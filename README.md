Уеб базирана система за управление на къщи за почивка.
Система за управление и показване на къщи за почивка с пълен административен панел. Позволява на администраторите да управляват къщи, населени места, типове обекти и потребители, докато обикновените потребители могат да разглеждат наличните опции.
